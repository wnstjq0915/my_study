## Python
### 반복문
명령어 | 기능
----|------
continue | 반복문의 처음으로 돌아감
break | 반복문을 멈춤
pass | 반복문을 빠져나감

#### while문
```python
i = 0
while i < 10
    """
    실행문
    """
```
- 조건반복 특화
- 무한반복 가능

#### for문
```python
for i in range('반복할 횟수') # range 대신 iterable 값을 넣어도 됨
    """
    실행문
    """
```
- 횟수반복 특화
